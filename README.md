# 💰 Money Manager App

A web-based Money Manager application built using HTML, CSS, and JavaScript (OOP).  
This app helps users track income and expenses, manage transactions, and view financial summaries efficiently.

---

## 📌 Project Overview

The Money Manager App is designed to help users manage their personal finances by:

- Tracking income and expenses
- Categorizing transactions
- Viewing financial summaries
- Performing CRUD operations on transactions

The application uses Object-Oriented Programming (OOP) principles and stores data using browser localStorage.
